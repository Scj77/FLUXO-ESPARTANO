# Layout da Nova Página - Análise da Imagem

## Estrutura Geral
A imagem mostra um layout em forma de diagrama com as seguintes seções:

### Seção Superior (RESUMO)
- **SI** (Saldo Inicial)
- **SF** (Saldo Final)

### Seção do Meio (CÁLCULO)
- **FLUXO** = DIFERENÇA - Renovações - Novos Clientes + Entradas Diversas - Saídas Diversas
- **DIFERENÇA** = SI - SF
- **P/NP?** (Pergunta se é Positivo/Negativo)

### Seção de Resultado
- **SUPER DÉFICIT** (se fluxo > 0 = SUPERÁVIT, se fluxo < 0 = DÉFICIT)

### Visualização Gráfica
- **Gráfica (pizza)** - Gráfico em pizza dos componentes

### Botões de Exportação
- **PDF**
- **JPG**
- **EXCEL**

### Resumo Total
- Lado direito com resumo total de todos os dados

## Fórmula do Fluxo
```
FLUXO = DIFERENÇA - Renovações - Novos Clientes + Entradas Diversas - Saídas Diversas
```

Onde:
- DIFERENÇA = SI - SF
- Se FLUXO > 0: SUPERÁVIT
- Se FLUXO < 0: DÉFICIT
- Se FLUXO = 0: EQUILIBRADO

## Dados que virão do Index
- Saldo Inicial (SI)
- Saldo Final (SF)
- Renovações (total)
- Novos Clientes (total)
- Entradas Diversas (total)
- Saídas Diversas (total)
